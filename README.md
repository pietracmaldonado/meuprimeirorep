# meuprimeirorep
## pietra 
# guia do github 
## 1 passo, criar um rep
## 2 passo, copiar o link do rep
## 3 passo, criar uma nova pasta "git"
## 4 passo, abrir no terminal a pasta "git"
## 5 passo, escrever no terminal "git clone *link do rep*"
## 6 passo, digitar "dir" para mostrar o conteudo da pasta 
## 7 passo, digitar "cd . *nome do rep*"
## 8 passo, digitar "code .", ira abrir o vs code
## 9 passo, digitar "git add . "
## 10 passo, digitar "git commit -m *alteração feita*"
## 11 passo, digitar "git push -u origin main"
